Physiotherapy on Lakeshore - Website

Project Info:

  • Created by:
    • Tristan Secord
  • Completed:
    • January 2015
  • Skills:
    • Objective-C / XCode
  • Client:
    • Physiotherapy on Lakeshore

Physiotherapy on Lakeshore is a small physiotherapy clinic located in Port Credit, Ontario. In order to compete with the larger physiotherapy chains with more resources, Physiotherapy on Lakeshore decided to market their clinic with a website. The website is visually appealing, while outlining all available services including Physiotherapy, Yoga, Acupuncture and more. Furthermore, the website introduces the team working at the clinic, providing a description on each members personality, responsibilities, and qualifications. Finally, the website allows for easy to use appointment booking, linked to their already functioning system.

This application was programmed mainly in HTML and CSS with some Javascript and jQuery for animations. It was written using Sublime Text and hosted using GoDaddy. Check out the website now.

Appointment Booking was one required feature of the website. Fortunately, the existing system in place for appointment booking was not too complicated, being done through email. This feature was accomplished using SendGrid email and bootstrap field validation.

In order to complete this contract, I first had to learn Objective-C. This was done over many hours of Lynda videos, and iTunes tutorials from Stanford University.
