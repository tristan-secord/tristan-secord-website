Nicole Lewis Photography - Website

Project Info:

  • Created by:
    • Tristan Secord
  • Completed:

  • Skills:
    • HTML / CSS / Javascript / jQuery / Sublime Text
  • Client:

Nicole Lewis is an entrepreneur in the photography business. Nicole Lewis Photography emphasizes photos of children commonly portrayed in grayscale. The website was designed to be advertise Nicole Lewis' previous work to bring in new perspective clients.

The website was mainly programmed in HTML and CSS with some Javascript and jQuery for animations.

Some additions on the backlog for this website are: Lazy loading images in the gallery for a faster response time, and designing a content managemnet system for Nicole to upload images with the click of a button.
