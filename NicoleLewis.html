Nicole Lewis Photography - Website

Project Info:

  • Created by:
    • Tristan Secord
  • Completed:
    • February 2016
  • Skills:
    • HTML / CSS / Javascript / jQuery / Sublime Text
  • Client:
    • Nicole Lewis Photography

Nicole Lewis is an entrepreneur in the photography businesss. Nicole Lewis Photography emphasizes photos of children portrayed in grayscale. The website is a marketing tool, designed to be simple and easy to use while displaying some of her best works from her vast portfolio.

The website was mainly programmed in HTML and CSS with some Javascript and jQuery for animations. The website was written using Sublime Text and hosted using GoDaddy. Check out the website now.

Some additions on the backlog for this website are: Lazy loading images in the gallery for a faster response time, and designing an admin page for Nicole to upload images with the click of a button.
